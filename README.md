Otel Rezervasyon Otamasyonu

Bu proje, SQL Server ile birlikte çalışarak bir otel rezervasyon sistemini yönetmek amacıyla geliştirilmiştir. Visual Studio kullanılarak C# dili ile yazılmıştır.

Özellikler
Bu proje şunları yapabilir:
- Özellik 1: Müşteri kaydı oluşturma ve silme.
- Özellik 2: Kayıtları güncelleme.
- Özellik 2: Veritabanına müşteri kaydetme.
- Özellik 3: Kaydı oluşmuş müşterileri sorgulama yöntemiyle arama.


Kullanılan Teknolojiler

Dil: C#

IDE: Visual Studio

Framework: .NET Framework / .NET Core

Veritabanı: SQL Server


Kurulum ve Çalıştırma
Projeyi bilgisayarınızda çalıştırmak için:

1.  Bu repoyu klonlayın veya ZIP olarak indirin.
2.  Klasörün içindeki `.sln` uzantılı dosyayı Visual Studio ile açın.
3.  (Varsa) Veritabanı bağlantı ayarlarını yapın.
4.  "Start" tuşuna basarak projeyi derleyin ve çalıştırın.
